변수의 선언규칙
1. 대소문자를 구분
2. 영문, $, _로 시작할 수 있다
3. 영문, $, _는 포함 할 수 있지만, 그 외에 특수문자는 사용할 수 없다
4. 키워드 및 예약어는 사용이 불가능하다
변수의 범위
1. 전역번수 : window 객체에 포함되는 변수, 다른 함수 등에서 공용으로 사용할 수 있다
값이 유지 된다
2. 지역변수 : 함수나 객체에서 선언되면 실행이 종료되는 시점에서 사라짐
변수의 저장과 선언
1. var / let / cont의 키워드를 통해서 **동적변수**로 저장된다
2. const를 통해서 상수의 값을 저장한다
3. javascript에서 변수의 타입은 동적으로 타입이 정해진다
javascript의 타입 종류
  • 문자(string)
  • 숫자(number)
  • 논리(boolean)
  • 널(object/null)
  • 객체(object/array)
  • **함수(function)
